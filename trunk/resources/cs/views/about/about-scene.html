MojoTracker na Google Code

Tento program je šířen za podmínek BSD Licence, jestli vám v MojoTrackeru chybí nějaká vlastnost, neváhejte a přispějte. Vývoj pro platformu webOS je velice jednoduchý!

Pokud najdete chybu, nebo máte nápad jak aplikaci vylepšit, vyplňte prosím issue.

Changelog

  • 0.2.6
    • Change track ordering
    • Fix for webOS 2
  • 0.2.5
    • Fixed map drawing for south hemisphere
  • 0.2.4
    • Make possible rename track
    • Fixed waypoint list handling on "info" page
  • 0.2.3
    • Draw waypoints on map preview
    • Fix timezone handling
  • 0.2.2
    • Add option for export waypoints (to *.loc file)
    • Add map preview to info page (use OpenStreetMap.org exports)
  • 0.2.1
    • Split export files is no longer needed
    • Reduce nodes count when device isn't moving
  • 0.2.0
    • Add waypoints support
    • Prevent sleep when tracking is on
    • Make MojoTracker translatable
    • Add popup menu to location (copy to clipboard, share via SMS/email)
    • Fixed export to kml (null values issue)
  • 0.1.29
    • Added screen with detailed track info
    • Added altitude and speed graphs
    • Added export to kml data format
  • 0.1.28
    • Warning: DB format isn't compatible with previous release
    • Displays and stores more information like altitude, position accuracy, speed...
    • Added preference page, specify preferred units.
    • Added page with list of saved tracks
    • Added option to delete saved track
    • Added export button, FileMgr service is required for this functionality (tested with FileMgr 0.5.1 api)
    • Added compass
  • 0.1.27
    • First public release
    • Display position information from internal GPS
    • Store tracks to SQLite file on internal storage
    • Exports tracks to gpx file - requires external conversion tool
